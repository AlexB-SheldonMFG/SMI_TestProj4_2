Power Source Requirements

When selecting a location for the unit, verify each of the following requirements is satisfied.

 

Model Amperage Model Amperage
SMI2 4.5 SMI7 6.5
SMI6 6.0 SMI11 10.0
SMI12 6.0*    
Model Amperage Model Amperage
SMI2-2 3.0 SMI7-2 5.0
SMI6-2 4.0 SMI11-2 5.0
SMI12-2 4.0*    

*The SMI12 comes with two power cords. It requires 6.0 amps for each cord.

*The SMI12-2 comes with two power cords. It requires 4.0 amps for each cord.

 

  • The required circuit-breakers are: SMO10HP-2 30 amps — SMO28HP-2 60 amps.
  • Each incubator is provided with a 125-volt, 15-Amp, 9ft 5 in (2.86m) NEMA 5-15P power cord. Always use this cord or an identical replacement.
  • Each unit is provided with a 230-volt, 10-Amp, EUR16P, 2.5 meters (8ft 2in), CEE 7/7 power cord. Always use this cord or an identical replacement.

  • SMI2: 250V T6.3 amp, 5x20mm
  • T3 amp, 250V, 5x20mm

 

  • Fuse Type: Two 2 amp, 250V, 5x20mm fuses
